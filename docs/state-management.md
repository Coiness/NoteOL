# 状态管理 (State Management)

## 功能概述

状态管理模块使用 Zustand 提供全局状态管理，支持用户会话、应用设置等全局状态。

## 实现方式

### 技术栈

- **Zustand**: 轻量级状态管理库
- **TypeScript**: 类型安全
- **React Context**: 局部状态管理

### 核心功能

1. **全局状态**: 用户信息、应用设置等
2. **状态持久化**: 本地存储状态恢复
3. **状态同步**: 跨组件状态共享

## 重要文件

### 状态文件

- `store/useStore.ts`: Zustand 状态存储
- `components/providers/providers.tsx`: React Context 提供者

## 亮点

1. **轻量级**: 相比 Redux 更简单的 API
2. **类型安全**: 完整的 TypeScript 支持
3. **性能优化**: 自动的状态订阅优化

## 注意事项和坑

1. **状态一致性**: 异步操作的状态更新
2. **内存泄漏**: 状态订阅的清理
3. **序列化**: 复杂对象的本地存储序列化
