# 测试与模拟 (Testing & Mocking)

## 功能概述

测试与模拟模块提供单元测试、集成测试和 API 模拟功能，确保代码质量和可靠性。

## 实现方式

### 技术栈

- **Vitest**: 测试框架
- **@testing-library/react**: React 组件测试
- **MSW**: API 模拟
- **jsdom**: DOM 环境模拟

### 核心功能

1. **单元测试**: 组件和工具函数测试
2. **集成测试**: API 和数据库集成测试
3. **API 模拟**: 开发时的 API 响应模拟
4. **测试覆盖率**: 代码覆盖率报告

## 重要文件

### 测试文件

- `vitest.config.ts`: 测试配置
- `vitest.setup.ts`: 测试环境设置
- `mocks/`: API 模拟数据
- `*.test.tsx`: 测试文件

## 亮点

1. **快速反馈**: 快速的测试执行
2. **开发效率**: API 模拟加速开发
3. **代码质量**: 自动化测试保障

## 注意事项和坑

1. **测试维护**: 测试用例随代码变化更新
2. **异步测试**: 异步操作的正确测试
3. **DOM 模拟**: jsdom 与真实浏览器的差异
